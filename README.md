# Sales ETL Pipeline – Python & SQL

## Project Overview
This project implements an end-to-end ETL pipeline that ingests raw sales data from CSV sources, performs data cleaning and transformation, and loads the processed data into a PostgreSQL database for analytics.

## Tech Stack
- Python (Pandas)
- PostgreSQL
- SQLAlchemy
- SQL

## Pipeline Workflow
1. Extract raw sales data from CSV files
2. Clean and transform data using Pandas
3. Load transformed data into PostgreSQL
4. Run analytical SQL queries for business insights

## Key Insights
- Monthly revenue trends
- Top customers by total spend
- Overall sales performance metrics
